UNDER CONSTRuCTION
==============

A rust server and a webpack frontend to server as lab for the
[Idionautic](https://github.com/navicore/idionautic-agent) project.

Setup
-----

```bash
cd static
ln -s ../../idionautic-agent/pkg agent
```


```bash
cd frontend
npx webpack --watch
```

```bash
cargo run
```

Then open http://localhost:3000
